# Overrun 🚧 **Status:** Active Development

## Overview

Overrun is a Unity-based action game currently under development. The project focuses on creating engaging combat mechanics, intelligent enemy AI, and smooth player interactions while following clean game architecture and scalable design principles.

---

## Features

- Player movement and camera controller
- Enemy AI with tracking and attack behaviour
- Shooting and combat mechanics
- Health and damage system
- Collision detection
- Interactive UI
- Modular game architecture

---

## Tech Stack

- Unity
- C#
- Unity Input System
- Physics Engine
- Visual Studio
- Git
